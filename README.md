### About

Software engineer based in Nairobi. I build backend systems for development finance by day, and a crypto-to-M-Pesa payments platform on the side.

- Currently **Assistant Software Engineer at [FSD Africa](https://fsdafrica.org)**, where I own the backend of DIMIS — a multi-tenant ASP.NET and SQL Server reporting platform used by 500+ staff and partners across FSD Africa, FSD Ethiopia and the Nyala Facility.
- Founder and engineer of **[CPay](https://cpay.co.ke)** — pay any Safaricom Paybill or Till directly from a crypto balance. Django, PostgreSQL, Redis, Celery, React Native. Live in private beta.
- Built **Daisy**, FSD Africa's internal agentic AI assistant, on an in-house RAG stack in n8n and Python.
